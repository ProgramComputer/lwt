Detailed Description

Show test frame.

Call: do_test_test.php?type=[testtype]&lang=[langid] Call: do_test_test.php?type=[testtype]&text=[textid] Call: do_test_test.php?type=[testtype]&selection=1
(SQL via $_SESSION['testsql'])

PHP version 8.1
